**Title: Mobile-Based Live Location Tracking System for Transport Vehicles Without GPS** --- ### üåê Overview To enable real-time location tracking of market-sourced vehicles (without GPS hardware), we'll design a web-based solution using the driver's smartphone GPS and internet connection. This system allows transport managers to track shipments live by sending a simple browser link to drivers. --- ### üöÄ Key Features 1. **Driver Tracking Link:** Unique tracking link per trip 2. **One-Click Access:** Driver opens link and grants location permission 3. **Live Location Updates:** Location pushed to central server every 5-10 seconds 4. **Admin Dashboard:** View all active drivers and their real-time positions 5. **Trip ID or Vehicle Number Mapping** 6. **Auto Expiry:** Tracking link auto-disables after X hours 7. **Mobile-Friendly UI** 8. **WhatsApp/SMS Link Sharing** 9. **Optional PWA (Progressive Web App) Support:** Allows persistent background tracking even if browser tab is closed 10. **Clickable Dashboard Tools:** Admin UI includes buttons to: - Create new
